## 1. 模块概述

**模块名称**：GoModParser - JSON流解析器  
**核心功能**：解析 `go list` 命令输出的JSON流，提取Go模块依赖信息  
**测试目标**：确保各种输入格式都能正确解析

## 4. 测试数据样例

### 4.1 基础模块JSON
```json
{
  "Path": "github.com/gin-gonic/gin",
  "Version": "v1.9.1",
  "Indirect": false,
  "Dir": "/home/go/pkg/mod/github.com/gin-gonic/gin@v1.9.1"
}
```

### 4.2 带Replace的模块
```json
{
  "Path": "github.com/old/pkg",
  "Version": "v1.0.0",
  "Replace": {
    "Path": "../local/pkg",
    "Dir": "/home/user/local/pkg"
  }
}
```

### 4.3 复杂嵌套模块
```json
{
  "Path": "github.com/complex/module",
  "Version": "v2.0.0",
  "Replace": {
    "Path": "github.com/local/fork",
    "Version": "v2.0.1",
    "Dir": "/home/user/local/fork",
    "GoMod": "/home/user/local/fork/go.mod"
  },
  "Require": [
    {"Path": "golang.org/x/crypto", "Version": "v0.17.0"}
  ]
}
```
