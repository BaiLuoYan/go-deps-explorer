# Changelog

## [0.1.0] - 2026-03-03

### Added
- Initial release
- Dependency tree view in Explorer sidebar
- Direct and indirect dependency grouping
- Dependency package directory browsing (lazy-loaded)
- Read-only file viewing via custom URI scheme
- Cmd+Click jump tracking and tree auto-reveal
- Multi-project workspace support
- Auto-refresh on go.mod changes
- Manual refresh button
- Configuration: handleReplace, showIndirect, vendorFirst
- Tooltip with dependency details (version, URL, replace info)
